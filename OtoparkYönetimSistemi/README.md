# Otopark Yönetim Sistemi

C# programlama dili kullanılarak geliştirilmiş basit bir konsol tabanlı otopark yönetim sistemi projesidir.
Kullanıcılar araç ekleyebilir, araç çıkarabilir ve otoparktaki araçları listeleyebilir.


# Özellikler

* Araç ekleme sistemi
* Araç çıkarma sistemi
* Otopark durumu görüntüleme
* Kapasite kontrolü
* Aynı aracın tekrar eklenmesini engelleme
* Konsol tabanlı menü sistemi
* Nesne tabanlı programlama yapısı
* Liste kullanımı


# Kullanılan Teknolojiler

* C#
* .NET Console Application


# İçerik

Projede aşağıdaki bölümler bulunmaktadır:

* Araç sınıfı
* Otopark sınıfı
* Program sınıfı
* Menü sistemi
* Araç ekleme işlemi
* Araç çıkarma işlemi
* Otopark durumu görüntüleme

Araç özellikleri:

* Plaka bilgisi

Otopark özellikleri:

* Kapasite bilgisi
* Araç listesi

Sistem içerisinde:

* Kullanıcıdan veri alma
* Liste yönetimi
* Döngü kullanımı
* Koşul yapıları
* Metot kullanımı
* Nesne oluşturma işlemleri

yer almaktadır.

# Amaç

Bu proje:

* C# sınıf yapısını öğrenmek
* Constructor kullanımını öğrenmek
* List koleksiyonlarını kullanmak
* Nesne yönelimli programlama mantığını kavramak

konularında pratik yapmak amacıyla geliştirilmiştir.
