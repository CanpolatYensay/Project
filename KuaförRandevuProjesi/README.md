# Kuaför Randevu Projesi 

HTML, CSS ve JavaScript kullanılarak geliştirilmiş modern bir kuaför randevu web sitesi projesidir.
Kullanıcılar online randevu oluşturabilir, hizmetleri görüntüleyebilir ve fiyat bilgilerine ulaşabilir.

# Özellikler

- Online randevu oluşturma
- Tarih ve saat seçme sistemi
- Hizmet ve fiyat listesi
- Firebase Firestore veritabanı desteği
- Admin paneli ile randevu yönetimi
- Randevu silme işlemi
- Responsive tasarım

# Kullanılan Teknolojiler

- HTML5
- CSS3
- JavaScript
- Firebase Firestore

# İçerik

Sitede aşağıdaki bölümler bulunmaktadır:

- Ana sayfa
- Randevu oluşturma alanı
- Hizmetler bölümü
- Fiyat listesi
- Hakkımızda bölümü
- Admin paneli

Örnek hizmetler:

- Saç Kesimi
- Sakal Tıraşı
- Saç + Sakal
- Çocuk Tıraşı
- Saç Boyama
- Damat Tıraşı

Her hizmet kartında:

- hizmet adı
- fiyat bilgisi
- açıklama bilgisi

yer almaktadır.

# Amaç

Bu proje:

- HTML form yapısı oluşturma
- CSS ile modern tasarım yapma
- JavaScript ile veri yönetimi
- Firebase Firestore kullanımı
- Responsive web tasarımı
- Admin panel sistemi geliştirme

konularında pratik yapmak amacıyla geliştirilmiştir.
