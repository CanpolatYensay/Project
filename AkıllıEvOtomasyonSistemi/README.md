# Akıllı Ev Otomasyon Sistemi

## Proje Hakkında

Akıllı Ev Otomasyon Sistemi, nesne yönelimli programlama (OOP) prensipleri kullanılarak geliştirilmiş bir C# konsol uygulamasıdır. 
Sistem üzerinden evde bulunan çeşitli cihazlar yönetilebilir ve durumları görüntülenebilir.

Bu proje; kalıtım (Inheritance), kapsülleme (Encapsulation) ve sınıf yapılarının kullanımını göstermek amacıyla geliştirilmiştir.


## Özellikler

### Işık Kontrolü
- Işığı açma
- Işığı kapatma
- Parlaklık seviyesini değiştirme
- Mevcut durumu görüntüleme

### Isıtıcı Kontrolü
- Isıtıcıyı açma
- Isıtıcıyı kapatma
- Mevcut durumu görüntüleme

### Kamera Kontrolü
- Kamerayı açma
- Kamerayı kapatma
- Mevcut durumu görüntüleme


## Kullanılan Teknolojiler

- C#
- .NET
- Object-Oriented Programming (OOP)
- Console Application
