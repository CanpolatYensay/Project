# Kebapçı Sitesi

HTML ve CSS kullanılarak geliştirilmiş basit bir restoran tanıtım web sitesi projesidir.
Kullanıcılar menüyü görüntüleyebilir, ürünleri inceleyebilir ve fiyat bilgilerine ulaşabilir.

## Özellikler

* Restoran ana sayfası
* Menü listeleme
* Ürün kartları
* Fiyat bilgileri
* İçindekiler açıklaması
* Görsellerle ürün tanıtımı
* Responsive düzen için CSS tasarımı

## Kullanılan Teknolojiler

* HTML5
* CSS3

## İçerik

Sitede aşağıdaki bölümler bulunmaktadır:

* Ana sayfa
* Menü
* İçecekler
* Tatlılar

Örnek ürünler:

* Adana Kebap Dürüm
* Adana Kebap Porsiyon
* Ciğer Porsiyon
* Tavuk Şiş Dürüm

Her ürün kartında:

* ürün görseli
* ürün adı
* fiyat bilgisi
* içerik açıklaması

yer almaktadır.

## Amaç

Bu proje:

* HTML yapı kurma
* CSS ile stil verme
* Flexbox kullanımı
* Kart tasarımı oluşturma
* Görsel ve içerik yerleşimi

konularında pratik yapmak amacıyla geliştirilmiştir.
