# Üniversite Kişi Sistemi

Bu proje, C# dili kullanılarak geliştirilmiş basit bir üniversite kişi yönetim sistemi uygulamasıdır.

Uygulama üzerinden öğrenci ve akademisyen bilgileri sisteme eklenebilir, listelenebilir ve rapor oluşturulabilir.

## Proje Amacı

Bu proje aşağıdaki nesne yönelimli programlama kavramlarını uygulamak amacıyla geliştirilmiştir:

- Sınıf yapısı (Class)
- Kalıtım (Inheritance)
- Soyut sınıf (Abstract Class)
- Arayüz (Interface)
- Çok biçimlilik (Polymorphism)
- Kapsülleme (Encapsulation)

## Kullanılan Teknolojiler

- C#
- .NET Console Application
