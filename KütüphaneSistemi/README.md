# Kütüphane Sistemi 

C# programlama dili kullanılarak geliştirilmiş basit bir konsol tabanlı kütüphane yönetim sistemi projesidir.
Kullanıcılar kitap ekleyebilir, kitapları listeleyebilir ve kitap arama işlemleri yapabilir.

# Özellikler

- Kitap ekleme sistemi
- Kitap listeleme sistemi
- Kitap arama işlemi
- Otomatik ID oluşturma
- Konsol tabanlı menü sistemi
- Nesne tabanlı programlama yapısı
- Liste kullanımı

# Kullanılan Teknolojiler

- C#
- .NET Console Application

# İçerik

Projede aşağıdaki bölümler bulunmaktadır:

- Kitap sınıfı
- Program sınıfı
- Menü sistemi
- Kitap ekleme işlemi
- Kitap listeleme işlemi
- Kitap arama işlemi

Kitap özellikleri:

- Kitap adı
- Yazar adı
- Sayfa sayısı
- Kitap ID bilgisi

Sistem içerisinde:

- Kullanıcıdan veri alma
- Liste yönetimi
- Döngü kullanımı
- Metot kullanımı
- LINQ ile arama işlemi

yer almaktadır.
