# BankaHesapSistemi

C# ile geliştirilmiş basit bir konsol uygulamasıdır. 
Kullanıcı banka hesabı üzerinde temel işlemleri terminal üzerinden gerçekleştirebilir.

## Özellikler

* Bakiye görüntüleme
* Para yatırma
* Para çekme
* İşlem geçmişini listeleme
* Menü tabanlı kullanım

## Kullanılan Teknolojiler

* C#
* .NET Console Application

## Uygulama Menüsü

Program çalıştırıldığında aşağıdaki seçenekler sunulur:

1- Bakiye Görüntüle
2- Para Yatır
3- Para Çek
4- İşlem Geçmişi
5- Çıkış

## Amaç

Bu proje nesne yönelimli programlama mantığını pekiştirmek amacıyla geliştirilmiştir.
Özellikle:

* Class yapısı
* Property kullanımı
* Metotlar
* Liste ile işlem geçmişi tutma
* Switch-case menü sistemi

konularını pratik etmek için hazırlanmıştır.
