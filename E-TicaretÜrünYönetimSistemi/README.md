# E-Ticaret Ürün Yönetim Sistemi

Bu proje, C# programlama dili kullanılarak geliştirilmiş basit bir E-Ticaret Ürün Yönetim Sistemi uygulamasıdır. Projede Nesne Yönelimli Programlama (OOP) prensipleri kullanılarak farklı ürün türlerinin yönetimi gerçekleştirilmiştir.

## Proje Amacı

E-ticaret sistemlerinde bulunan ürünlerin ortak özelliklerini tek bir temel sınıfta toplamak ve farklı ürün kategorilerini kalıtım (Inheritance) kullanarak yönetmektir.

## Kullanılan Teknolojiler

- C#
- .NET
- Object-Oriented Programming (OOP)
- Console Application

## Amaç

Bu proje sayesinde aşağıdaki konular pekiştirilmiştir:

- Soyut sınıflar (Abstract Classes)
- Kalıtım (Inheritance)
- Çok Biçimlilik (Polymorphism)
- Kapsülleme (Encapsulation)
- Constructor Kullanımı
- Override Metotlar
- Koleksiyonlar ve Nesne Yönetimi
