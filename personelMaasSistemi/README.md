# Personel Maaş Sistemi 💼

C# programlama dili kullanılarak geliştirilmiş nesne tabanlı bir personel maaş hesaplama sistemi projesidir.
Sistem içerisinde farklı personel türleri için maaş hesaplama işlemleri gerçekleştirilmektedir.

# Özellikler

- Personel sınıf yapısı
- Kalıtım (Inheritance) kullanımı
- Polymorphism kullanımı
- Maaş hesaplama sistemi
- Override metot kullanımı
- Personel bilgilerini görüntüleme
- Nesne tabanlı programlama yapısı

# Kullanılan Teknolojiler

- C#
- .NET Console Application

# İçerik

Projede aşağıdaki bölümler bulunmaktadır:

- Personel sınıfı
- Müdür sınıfı
- Maaş hesaplama sistemi
- Bilgi gösterme sistemi
- Constructor yapıları
- Virtual ve Override metotlar

Personel bilgileri:

- Ad
- Soyad
- Yaş
- Maaş bilgisi

Sistem içerisinde:

- Kalıtım kullanımı
- Encapsulation yapısı
- Virtual metotlar
- Override işlemleri
- Constructor kullanımı
- Console çıktıları

yer almaktadır.

# Amaç

Bu proje:

- C# OOP mantığını öğrenme
- Kalıtım yapısını kullanma
- Polymorphism mantığını anlama
- Virtual ve Override metotları kullanma
- Constructor yapıları oluşturma
- Maaş hesaplama sistemi geliştirme
- Konsol uygulaması geliştirme

konularında pratik yapmak amacıyla geliştirilmiştir.
